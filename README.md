# fco-2

## Trikotwäscheliste ⚽

Eine Webapp zur Verwaltung der Trikotwäsche für die Fußballmannschaft.

**[Zur Trikotwäscheliste](https://maikrehburg.github.io/fco-2/laundry/)**

### Features

- Spielerverwaltung
- Waschdaten mit Datum und Gegner/Anlass erfassen
- Übersichtliche Darstellung aller Einträge
- Daten werden im GitHub Repository gespeichert
- Lesen ohne Login möglich
- Schreibzugriff mit GitHub Token

### Verwendung

Die App ist öffentlich zugänglich. Zum Ansehen der Liste ist kein Token erforderlich. Zum Hinzufügen oder Bearbeiten von Einträgen wird ein GitHub Personal Access Token benötigt.
